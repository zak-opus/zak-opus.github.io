# 2021

一、云原生前端框架项目（0601-）

  • 开发一个具有中英文切换、自定义主题切换、网关加密、内置组件、请求封装等功能的项目，提交到 gitlab；

  • 抽离项目功能代码，封装成cnap-ui@0.2.0插件，并上传 npm 仓库；

  • 升级cnap-ui@0.2.1，优化组件风格与使用；

  • 部署插件测试项目到远程服务器用于预览插件效果；

  • 编写框架使用手册

二、开发 vue 大文件切片上传项目（1213-1222）

  • 借助 vue-simple-uploader 插件开发具有切片、断点、秒传等功能的文件上传实例项目，上传到 gitlab 仓库；

  • 使用 spark-md5 生成文件唯一标识；

  • 编写项目使用手册。

三、开发 sm24 加密性能对比项目（1220-）

  • 模拟文本使用 sm2\sm4 进行加解密，并将加解时间分别通过 echarts 图表展示；
